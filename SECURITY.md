# Security Policy

## Supported Versions
This is a learning/demo project. Only the latest version on the `main` branch and the latest tagged release are supported.

## Reporting a Vulnerability
If you believe you have found a security vulnerability:
1. Please do **not** open a public issue.
2. Instead, open a **private GitHub Security Advisory** for this repository (preferred), or contact the maintainer privately.

Thank you for helping improve the security of this project.
